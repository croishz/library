접근 가능한 이름 "Label in name"

aria-label

  • title 속성과 동일.
  • text는 간결해야 함. 설명은 aria-desciptedby를 사용.
  • 사용자에게 전달할 내용이 텍스트가 시각적으로 보이면 aria-labelledby,
    아이콘등으로 처리하여 보이지 않으면 aria-label을 사용한다. [wcag 권장안]
  • screen reader가 읽지 못하는 경우가 있음.
    • ex/ p, i, u, del, span, div : aria-label이 아닌 text를 읽음.
